# My Portfolio Site 🚀

Personal portfolio website for presenting my frontend work, project examples, and contact options to potential clients or employers.

## 👋 About

I am Ilya, a junior frontend developer focused on building responsive landing pages, clean web interfaces, and small web products with React, Next.js, and TypeScript.

This site is designed to show:

- 💼 services I can provide
- 🧩 my development process
- 🖼️ live project previews
- 📩 a contact form for requests
- 🌍 multilingual content

## ✨ Features

- 🎬 Hero section with video background
- 🧭 Section navigation
- 🌐 Language switcher
- 🧱 Services and process sections
- 🖼️ Interactive project carousel
- 🔄 Flip cards with project descriptions
- 📬 Contact form
- 📱 Responsive layout

## 🛠️ Tech Stack

- Next.js
- React
- TypeScript
- Tailwind CSS
- Vercel-ready deployment

## 🚀 Getting Started Locally

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open:

```text
http://localhost:3000
```

## ✅ Quality Checks

Run lint:

```bash
npm run lint
```

Build for production:

```bash
npm run build
```

## 📌 Purpose

This portfolio is built for clients and employers who want to quickly understand what I can create, how I work, and how to contact me for a project or collaboration.
